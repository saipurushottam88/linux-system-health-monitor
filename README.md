# Linux System Health Monitoring Tool (Bash)

A Bash-based monitoring script that tracks system performance and detects potential resource issues in Linux environments.

## Features
- Monitors CPU usage using `top`
- Checks memory usage using `free`
- Tracks disk utilization using `df`
- Generates threshold-based alerts for high disk usage

## Technologies Used
- Bash (Shell Scripting)
- Linux Core Commands

## How to Run
1. Make script executable:
   chmod +x monitor.sh
2. Run:
   ./monitor.sh
